## 大事小情

- 十九个小时的飞机坐起来竟然没有想象中可怕，可能是因为选了靠过道的位子可以经常起来，但真的也飞太久了！binge-watch 了一整季《真探》感觉对时间的感知都模糊了，简直不知道过了多久
	- 但是海航的飞机餐有点过于难吃了，在我吃过的所有飞机餐里也是数一数二的难吃，提醒自己回程的时候得带点零食上飞机……
- 回来之后到现在的作息不可不谓健康，竟然一直能保持在十二点前睡觉，可喜可贺，可喜可贺
	- 试着用了用重力眼罩，竟然这么好用，戴上可以光速睡着；不过感觉这东西有点依赖性
- 跟着学了一个 Claude Code 机制的 GitHub 项目 [learn-claude-code](https://github.com/shareAI-lab/learn-claude-code)
- 开始上班了，差点被早高峰的六号线挤个半死
	- 上班的一个附带好处（？）是增加了看书的时间，现在每天在地铁上一来一回能看俩小时书，因此这期的读书量又上来了。
- 又开始用微信读书了，它的多端同步确实非常香；虽然现在远不如之前那么猛送阅读卡了，不过外部导入书籍没有什么限制弥补了这一点
	- 忘了在哪儿看到过微信读书好像会自动和谐导入的书籍，呃，试试再说吧。
- 把备忘录上要吃的东西吃了 1/3
## 书影音演出

### 书

{{< douban-card "https://book.douban.com/subject/3277199/" >}} 

这真的很像 BL 作品，很可惜两个主角都不是我喜欢的类型，而这种便利屋单元故事如果读者无法喜爱主角基本就弱了一半。更好奇的是为什么妓女、黑帮、毒品这些元素能如此顺滑地融入琐碎的现代城市生活（

{{< douban-card "https://book.douban.com/subject/36096287/" >}} 

都市生活的迷人之处尽在八平米之外，读完确实也产生了“应该多出门走走”的想法，虽然这种生活我肯定是过不了……

{{< douban-card "https://book.douban.com/subject/37271274/" >}} 

萌萌的很热闹！我此前只看过一部《子弹列车》，甚至不是原作小说，但看了感觉应该把杀手系列的前文补一补。虽然是这个题材，但因为刻意写得脱离现实，几乎像一种江湖题材了，非常轻盈明快，而且还是我非常喜爱的酒店封闭空间题材，场景转换叠加在多线叙事上，格外流畅。

{{< douban-card "https://book.douban.com/subject/6889894/" >}} 

别姬小姐最终作，虽然还是没死什么人，但最后十页，最后一句话完全把前面那种平静、精巧甚至于有些无聊的气氛一扫而空，大为震撼，而这还是我对日本近代史几乎一无所知，恐怕一多半内容都没读出来的前提下……总之，这本拿直木奖都有点委屈它了，应该去碰一下芥川奖的。

{{< douban-card "https://book.douban.com/subject/4152750/" >}} 

诡计是很精彩的，关键是猎奇描写除了感官刺激之外毫无意义，非常没劲。

{{< douban-card "https://book.douban.com/subject/30475757/" >}} 

不应该刚看完《杀戮之病》就看这本的，嗯……虽然我对中间的若干细节不是十分感冒，不过诡计还是处理得相当自然。

{{< douban-card "https://book.douban.com/subject/37444748/" >}} 

发现之前看过最后一篇（《被砍了头之后也不死的我们的无头杀人事件》）的讲解视频，所以失去了一些初见的新鲜感，这应该也是最无厘头（字面）和精神状态最超前的一篇……第一篇也蛮有趣的，这个如何在死前十五秒内打出最高伤害的点子很好玩，而且能把如此电波的设定系写得这么流畅也非常不容易。

{{< douban-card "https://book.douban.com/subject/35602671/" >}} 

虽然我喜欢伊坂的多线叙事，但这本的故事本身并不是很有趣，好在还是塑造了黑泽这么一个比较有趣的角色。

{{< douban-card "https://book.douban.com/subject/10554308/" >}} 

终于看了这本（其实东野最有名的好些书我都没看过，一味地读了一堆伽利略……），两个主角从头到尾没有明面的互动也没有正面描写，一切都从旁人眼光中浮现，前后细节一点一点勾连出故事的全貌，在 500 多页的篇幅里高潮迭起，没有什么冗余篇幅，极为不易。

{{< douban-card "https://book.douban.com/subject/26877752/" >}} 

反转还不错，但是尽管相信恶意能驱使人做这么多事情，动机仍然不是很能打动我。

{{< douban-card "https://book.douban.com/subject/26681984/" >}} 

多视角很难把每个人的心理都把握得比较准确，凑佳苗做到了，本书堪称东亚家庭神经病之大全。

{{< douban-card "https://book.douban.com/subject/36743030/" >}} 

谁改的这个破书名啊！谁啊！挺喜欢这本的感觉，特别是梨帆部分的剧情，一看豆瓣短评竟然有人认为现实这条线的连篇累牍的出版界日常很无聊，真是吃不到一个碗里……倒是觉得小说线里有些设置稍显刻意了一些。

### 影

{{< douban-card "https://movie.douban.com/subject/25824851/" >}} 

相当 old school，还不错，剧情很扎实，就是有点平淡，感情戏也非常 old school 地不如没有……。比较喜欢警探剧里有一些办公室政治的戏份。

{{< douban-card "https://movie.douban.com/subject/35809047/" >}} 

在回国的飞机上一口气看完了。朱迪福斯特演什么都很有魅力，但不得不说我很不欣赏这种把传统警探剧男主角直接性转的做法，相当偷懒，剧情本身到后期也是如此，推进得十分潦草，可惜了前面的极地科考站设定还挺有吸引力的。

{{< douban-card "https://movie.douban.com/subject/1291990/" >}} 

也是在飞机上看的，长对白片子的好处是看起来时间过得很快，坏处是——至少对这个系列而言，两个主角我都不怎么感冒，对话中常常出现一些我很不喜欢的东西……调度还是很好的。

{{< douban-card "https://movie.douban.com/subject/36402972/" >}} 

好像吃了一万篇 ao3 同人文写出来的，这话不是夸的意思，能明显看出把六集缩成了一集导致的仓促，情节衔接经常没头没脑，可以说看完完全不记得中间发生了什么，尼尔盖曼你罪该万死啊……！怎么说好歹给了粉丝一个幸终，staff 辛苦了。
