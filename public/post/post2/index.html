latex

只在需要渲染 latex 的页面加载，在 frontmatter 设置 math: true 即可。显示单行方程式，用 $$ 作为分隔符，如：
$$y_t = \beta_0 + \beta_1 x_t + \epsilon_t$$
这会在网页上渲染为：
$$y_t = \beta_0 + \beta_1 x_t + \epsilon_t$$

要显示 inline 方程，用 \\( \\) 分隔：
\\(y_t\\)
效果如 \(y_t\)。
