新标日初级上册小结

Posted by LiYixian on Monday, January 12, 2026 | 阅读 | 共 4567 字，阅读约 10 分钟

在胡密老师的笔记基础上增加了更详细的说明和例句。

一、用言

什么是用言？
有活用（即词尾变化，类似于变位 conjugation）的独立词，可以单独构成谓语。

1. 动词

动词变形

一类动词（五段动词）：词尾假名落在五十音图う段（如う、く、す等），或以「る」结尾但前接假名不在い段或え段。

二类动词（一段动词）：词尾是「る」，且倒数第二个假名必须位于「い段」（上一段动词，如見る）或「え段」（下一段动词，如食べる）。
注意存在符合二类动词结构特征但属于一类动词的特例（如「帰る」），需要单独记忆。

三类动词：只有两个，する、来る，都需要单独记忆。

动词变形规律：
ない形→ます形→辞書形（基本形）　词尾遵循 あいう 的变形顺序。

单词类别 ない形 ます形 辞書形 て形 た形
一类，く、ぐ 结尾 「う」段 →「あ」段 + ない 「う」段 →「い」段 + ます / い音变「いて/いで」
*注意：行く的て形比较特殊，为：行って		 い音变「いた/いだ」
一类，う、つ、る 结尾 「う」段 →「あ」段 + ない
*注意：う→わ		 「う」段 →「い」段 + ます / 促音变「って」 促音变「った」
一类，む、ぶ、ぬ 结尾 「う」段 →「あ」段 + ない 「う」段 →「い」段 + ます / 拨音变「んで」 拨音变「んだ」
一类，す结尾 「う」段 →「あ」段 + ない 「う」段 →「い」段 + ます / 「し」+ て 「し」+ た
二类 る→ない る→ます / る→て る→た
来る 来（こ）ない 来（き）ます くる 来（き）て 来（き）た
する しない します する して した

变形示例表：

单词 ない形 ます形 辞書形 て形 た形
書く 書かない 書きます かく 書いて 書いた
行く 行かない 行きます いく 行って 行った
急ぐ 急がない 急ぎます いそぐ 急いで 急いだ
買う 買わない 買います かう 買って 買った
待つ 待たない 待ちます まつ 待って 待った
売る 売らない 売ります うる 売って 売った
読む 読まない 読みます よむ 読んで 読んだ
飛ぶ 飛ばない 飛びます とぶ 飛んで 読んだ
死ぬ 死なない 死にます しぬ 死んで 死んだ
話す 話さない 話します はなす 話して 話した
食べる 食べない 食べます たべる 食べて 食べた
見る 見ない 見ます みる 見て 見た
来る 来（こ）ない 来（き）ます くる 来（き）て 来（き）た
する しない します する して した

各形态用法

简体形 敬体形
肯定 行く 行きます
否定 行かない 行きません
过去肯定 行った 行きました
过去否定 行かなかった 行きませんでした

ない形

  • ないでください　请别做
    • 部屋の鍵を忘れないでください。
  • なければなりません　一定要做
    • 李さんは今日早く帰らなければなりません。
  • なくてもいいです　不做也可以
    • 明日は残業しなくてもいいですよ。

ます形

  • ます→たいです　想做、希望
    • 私は映画を見たいです。
  • ます→ませんか　邀请、劝诱
    • 一緒にお茶を飲みませんか。
  • ます→ましょう　建议
    • ちょっと休みましょう。
  • ーます＋に＋运动动词　来去目的
    • 今日デパートへ買い物に行きます。
  • ます→ましょうか　有礼貌、疑问
    • 窓を閉めましょうか。
  • 构词法：ます→物・方・中　

辞書形

  • ＋ことができる　能做
    • スミスさんはピアノを弾くことができます。
      • 辞書形＋こと　形式体言，动词名词化（doing）
  • は〜ことです　是做……
    • 私の夢は先生になることです。
  • ＋前に　做…之前
    • こちらへ来る前に、電話をかけてください。

て形

  • て〜て〜　顺序、并列
    • 昨日デパートえ行って、買い物しました。
    • ホテルの部屋は広くて明るいです。
    • このコンピュータの操作は簡単で便利です。
  • てから　强调先后顺序，一…就…
    • 李さんは毎晩ラジオを聞いてから寝ます。
  • てください　请做
    • ここに住所と名前を書いてください。
  • ＋もいいです　可以做
    • ここで写真を撮ってもいいですか。
  • てはいけません　不允许做
    • 飛行機の中でタバコを吸ってはいけません。
  • て＋补助动词
    • います　正在做 / 持续状态
      • 小野さんは今新聞を読んでいます。
      • 森さんの車を持っています。

た形

  • ＋ことがある　曾经做过
    • 私はすき焼きを食べたことがあります。
  • ＋後で　做…之后
    • 李さんは会社が終わった後で、飲みに行きます。
  • ＋ほうがいいです　做…比较好
    • 静かなほうがいいです。
    • もっと野菜を食べたほうがいいですよ。
  • たり〜たり〜　有时…有时…/有些…有些…
    • 小野さんは休みの日、散歩したり買い物に行ったりします。
    • 昨日のパーティーで皆食べたり話したりしました。
    • 日本語の先生は中国人だったり日本人だったりです。
    • 〜によって　根据…的区别而…
      • 季節によって高かったり安かったりです。
      • 日によって暇だったり忙しかったりです。

自动词和他动词

自动词：不需要借助宾语，动词本身能完整地表达主语的某种动作。
例：なります 表自身变化

  • 形 1：子供が大きくなります。
  • 形 2：町は静かになります。
  • 名词：息子は医者になりました。

他动词：需要借助宾语才能完整地表达主语对对象施加的某种动作。
例：します 让…变化

  • テレビの音を大きくします。
  • 部屋をきれいにしてください。

2. 形容词

い形和な形

い形容词（一类形容词）：最后一个音节是「い」的形容词。
な形容词（二类形容词、形容动词）：最后一个音节是「だ」的形容词，但词典和单词表上经常省略だ（如 静かだ），既有形容词的意义，又有动词的活用变形。

形容词用法

变形规律：

い形简体 い形敬体 な形简体 な形敬体
肯定 忙しいです 忙しい 暇です 暇だ
否定 忙しくないです 忙しくない 暇ではありません 暇ではない
过去肯定 忙しかったです 忙しかった 暇でした 暇だった
过去否定 忙しくなかったです 忙しくなかった 暇ではなりませんでした 暇ではなかった

接续用法：

い形 な形
+名词 忙しい人 暇な人
+动词 忙しくなります 暇になります
て 忙しくて 暇で
たり 忙しかったり 暇だったり

用言用法

  • かどうか
    • 私は今年の夏、北京へ行くかどうか分かりません。
  • （疑问）从句
    • 鍵がどこにあるか教えてください。
    • 今日は来るかどうか分かりません。
  • と思います/言います
    • 李さんはもうすぐ来ると思います。
    • 「〜と思う」多用于第一人称，第三人称要用「思っています」
      • 小野さんはこの本が面白いと思っています。
    • 陳さんはパーティーに行くと言いました。
  • んです　强调事实，说明理由
    • すみません、頭が痛いんです。

二、体言

什么是体言？
没有活用的独立词，后续助词が（は、も等）构成主语。

1. 数词

这部分内容比较多和繁杂，应该会单独整理一篇，下面只是简略提及。

基数词

  • 0−10、百、千、万、億
  • 1-10 的训读

注意一些特殊的变形和规则：

100、1000 直接读「百ひゃく」，「千せん」，前面不加「一いち」，10000，100000000 则需要读作「一万いちまん」、「一億いちおく」。
300，3000 后面的「ひゃく、せん」会变为「びゃく、ぜん」。600，800 会有促音便，同时「ひゃく」会变为「ぴゃく」。8000 也会有促音变。
还有一点，如果数字中有 0，汉语中会读出 0，但日语中直接跳过，比如 108 读作「ひゃくはち」。
—— 日语的数字

序数词

第〜、〜目、〜番

助数词

  • 本、枚、冊
  • 音读和训读；音变
  • 在句中的位置

2. 代词

「こそあど」系列

词性 作用 こ系列 そ系列 あ系列 疑问词
连体词 修饰名词 この その あの どの
连体词 表示属性 こんな そんな あんな どんな
代词 指代事物 これ（ら） それ（ら） あれ（ら） どれ、なに（なん）
代词 指代人 この人/方 その人/方 あの人/方 だれ、どなた
代词 指代场所 ここ そこ あそこ どこ
代词 指代方向 こちら そちら あちら どちら

注意「ここ／そこ／あそこ」和「こちら／そちら／あちら」的区别。以下引用自 「こそあど」系列（二）：

「ここ／そこ／あそこ」用于表示场所，例如：

ここは図書館です。 // 这里是图书馆。

「こちら／そちら／あちら」一般用于表示方向。例如：

図書館はそちらです。 // 图书馆在那边。

「こちら／そちら／あちら」也可以作为「ここ／そこ／あそこ」的礼貌用语：

本はこちらにあります。 // 书在这里。
本はここにあります。 // 书在这儿。

「こちら／そちら／あちら」还可以用于指人，作用与「この／その／あの＋方」相同，都是比较礼貌的说法。

三、副词

频率

频率 单词 含义
90% いつも 总是
70% よく 经常
50% 時々（ときどき） 有时
30% たまに 偶尔
10% あまり〜ない 不太
0% 全然〜ない 完全不

程度

单词 含义
一番（いちばん） 有某种特定的范围，是其中最高（最低）的
すごく 非常
とても 非常，比较柔和的口语表达
大変 非常，表示对事情的惊讶
大抵（たいてい） 大致、大部分、通常，表示占比高
よく 好好地、仔细地
まあまあ 还算可以
ちょっと 稍微
まだ 还、尚未
もう （过去时）已经；（将来时）快要
本当に 真是、实在

四、助词

が

  • 动词要求
    • 分かります、できます、あります、います
  • 形 1 要求
    • 欲しい
  • 形 2 要求
    • 上手、好き、嫌い
  • 强调主语
    • 私が吉田です。
  • 表部分
    • 餃子は皮が美味しいです。
  • 自然景色
    • 雨が降ります。
  • 比较结果
    • 中国のほうが広いです。
  • 从句主语
    • 仕事が忙しいですから、帰りません。
  • 表转折
  • 缓和语气
    • ちょっと分からないんですが、教えてくださいませんか。

に

  • 存在
    • 部屋に机があります。
  • 具体时间
    • ６時に起きます。
  • 给的对象
    • 母に手紙を書きます。
  • 从…得到
    • 友達に辞書をもらいました。
  • 频率
    • 週に4 回学校へ行きます。
  • （来去的）目的
    • 食べに行きます。
  • 归着点
    • 部屋に入ります、バスに乗ります
  • 目的地
    • 病院に行きます。
  • 固定用法
    • 友達に会います。

を

  • 表宾语
    • ご飯を食べます。
  • 离开起点
    • バスを降ります。
    • 家を出ます。
  • 途径、经过的场所
    • 公園で散歩します。
    • このバスは駅前を通りますか。

へ

  • 方向

で

  • 方式方法
    • バスで会社へ行きます。
  • 动作场所
    • 学生は今教室で勉強しています。
  • 范围
    • 日本の季節の中で、秋が一番好きです。
  • 表状态
    • 1人で図書館へ行きます。

より

  • 比较
    • 李さんは王さんより有名です。
    • 王さんより、李さんのほうが有名です。
    • 神戸は大阪ほど賑やかではありません。

から

  • 从…，动作时间或起点
    • 昨日、何時から何時まで働きましたか。
    • 李さんは先月北京から来ました。
  • 表原因
    • 森さんはお酒が好きですから、毎日飲みます。

と

  • 名词并列（全部）
    • 机の上に本とノートがあります。
  • 共同做事的对象
    • 小野さんは友達と帰りました。
  • 指代说话、思考的内容
    • 明日はいい天気だと思います。

や

  • 名词并列（部分）
    • 吉田さんは時々中国や韓国へ行きます。

か

  • 表选择，或者
    • 私は毎朝パンかお粥を食べます。
  • 疑问词＋か 表不确定
    • 夏休み、どこかへ行きないです。

も

  • 也
  • 全面（否定）
    • 部屋に誰もいません。
  • 两种都：も〜も〜
    • ファックスもメールも届きました。

五、疑问词

  • 何（なに / なん）：什么（问物）
    • 先生は何が食べたいですか。
  • 誰（だれ / どなた）：谁（问人）
    • 誰が来ましか。
  • どこ / どちら：哪里
    • 李さんは学校はどこですか。
  • いつ / 何時：什么时候
    • いつ 问大概时间，何時 问具体时间
    • 夏休みはいつからですか。
  • いくら：多少钱
    • 靴はいくらですか。
  • 何歳 / おいくつ：多少岁
    • 森さんはおいくつですか。
  • どれ：哪个
    • どれが小野さんのコートですか。
  • どの + 名词：哪个
    • どのコートが小野さんのですか。
  • どうして：为什么
  • どう / いかが：如何，对形容词作谓语提问
    • 今日の天気はどうですか。
  • どんな：什么样的，对形容词+名词提问
    • 森さんはどんな人ですか。
  • どちら：哪个，二者比较
    • リンゴとバナナと、どちらが好きですか。
  • どのぐらい：程度，多少、多长时间
    • どのぐらい日本にいますか。
  • いくつ：多少个
    • リンゴはいくつありますか。
  • どうやって：怎么做
    • どうやって会社に行きますか。

六、连词

  • 顺接连词，而且、另外：それに
    • 彼は英語が話せます。それに、フランス語もできます。
  • 转折：が、でも、けど
    • ちょっと難しかったけど、まあまあできたよ。
  • 转换话题，对了、by the way：ところで
    • 明日は会議があります。ところで、先週の報告書はもう見ましたか。
  • 选择，还是、或者：それとも
    • 犬が好きですか。それとも、猫が好きですか。
  • 原因：だから / ですから
    • 今日は雨が降るよ。だから傘を持って行ってね。
